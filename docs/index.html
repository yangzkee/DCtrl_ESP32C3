公式原理

把两个连杆想成两根小棍子：它们能把末端送到离原点某个距离 r 的地方。最远是两根完全伸直，最近是长棍子折回来还多出的那一段。

最远距离 = A + B 最近距离 = |A - B| 所以可达条件： |A - B| <= r <= A + B

末端点是 (X, H)，它到原点的距离来自直角三角形：

r = sqrt(X^2 + H^2)

把两条公式合在一起，当前目标点可达的完整判断就是：

|A - B| <= sqrt(X^2 + H^2) <= A + B
因为 H 是负数，计算时用 H² 或 |H|，负号只表示方向在原点下方，不会改变距离大小。

高度固定时的推导

1

先看外圆。末端不能比 A+B 更远，所以 X² + H² <= (A+B)²。

2

把 H 固定住，只留下 X：X² <= (A+B)² - H²。

3

所以单侧最大横向位置是 Xmax = sqrt((A+B)² - H²)。

4

如果高度太靠近原点，还要避开内圆：Xmin = sqrt((A-B)² - H²)。这时中间一段不能到达。

可达与不可达情况

情况 条件 水平范围 含义
整条高度线不可达 |H| > A + B 没有 X 可以到达 高度比两根杆伸直还远。
只碰到一个点 |H| = A + B X = 0 两根杆必须竖直伸直。
连续可达 |A-B| <= |H| < A+B -Xmax <= X <= Xmax 末端可以在这条水平线上连续左右移动。
左右分段可达 |H| < |A-B| [-Xmax,-Xmin] 和 [Xmin,Xmax] 中间离原点太近，被内圆挡住。
Xmax = sqrt((A+B)^2 - H^2) Xmin = sqrt((A-B)^2 - H^2) 只在 |H| < |A-B| 时出现
